Version 0.9.9 brings important bugfixes and usability improvements.

Bugfixes

  • Fixed crash when connecting a GitHub account that does not contain repositories.
  • Fixed hang when pushing configuration changes.
  • Fixed possible hang when resolving merge conflicts.
  • Fixed possible race condition when selecting an existing GitHub repository.
  • Fixed asset dimensions occasionally appearing on folder rows.

Changes

  • Streamlined trial signup flow: from landing page, to mailing list confirmation, to getting started.
  • When adding assets to a repository without subfolders, Gemba no longer asks for the target folder but just copies them to the root folder.
  • Repositories with an "empty" remote, i.e. without a branch, are now supported.
  • Removed Google Analytics tracking.
