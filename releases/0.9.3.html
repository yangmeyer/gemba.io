New Features

  • Onboarding option to try Gemba out with a sample repo.
  • Added checks for @3x Retina HD assets.

Fixed Bugs

  • Dropping non-assets was possible, dirtying the working directory.
  • Show alert if local repositories don’t have a .gemba file.
  • Dropped .gemba file was silently ignored if it points to a repo that already has a .gemba file.
  • Crashed when loading a repository with no corresponding remote branch.

UI Fixes & Improvements

  • .gemba files now previewable system-wide with Quick Look.
  • Upon launch, the repo selection triangle was misplaced.
  • Show info if dropped .gemba file will be ignored (when remote .gemba file exists).

Misc Improvements

  • Gemba can recover from repositories being bisected.
  • Repository health check now done on refresh.
