Thanks for all the bug reports and feedback so far! I’ve addressed some of the issues in this release.

Bugfixes

  • User-provided summary was missing in commit message.
  • Discard while pulling led to errors and polluted local repo with a temp commit.

Improvements

  • Subsequent beta updates will be served through Sparkle.
  • Add an explanatory comment to the auto-generated .gemba file.

UI tweaks

  • Summary field now stays enabled while refreshing.
  • Append "(new)" to asset name, instead of "+" marker.
  • Improved switching between repos.
  • URL field is now disabled while cloning.

Known issues

  • Filenames containing umlauts (and probably other non-ASCII) seriously confuse Git, and subsequently, Gemba.
  • Missing user feedback after dragging in Unchanged asset.
  • The app icon is a placeholder.

If you have any questions at all, drop me a line at yang@gemba.io. Thanks!
