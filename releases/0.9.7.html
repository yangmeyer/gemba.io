New Features

  • Delivery-based trial replaces time-based trial.
  • Repository configuration panel (view/edit its .gemba file from the app).
  • Onboarding: New users are guided to their first delivery by a quest.
  • After connecting a repo, user gets to view its existing or create a new configuration.
  • Adding new assets now asks where to put them.
  • Xcode Asset Catalogs: Gemba can now create image sets for new assets.
  • Support adding local repository working copies.

Fixed Bugs

  • Fix cloning for users new to Git when SSH hadn't been used with hoster.
  • Fix inferring wrong remote when connecting an unconfigured repository with multiple remotes.

Changes

  • De-emphasized sample repository in onboarding process.
  • Bundle git binary for user convenience.
